## 🎨 Design Features

- **Light Mode Default with Dark Mode Toggle**: Responsive mode toggling with browser local storage state caching.
- **Symmetric Typography Formatting**: Built using Google Fonts (*Outfit* and *Inter*) with adjusted line-heights and sizing weights.
- **Visual Tech Badges & Icons**: Handpicked FontAwesome symbols styling each technology skill and project.
- **Interactive Model Sandbox**: Dashboard letting users adjust Confidence thresholds, IoU, and Data Augmentation levels to view predicted mAP/Accuracy and latency FPS outputs.
- **Direct-to-Gmail Contact Form**: Automated messages routing using Web3Forms endpoint posting.
- **Mock AI Chatbot Assistant Widget**: Fully functional resume Q&A chatbot with support for a live Gemini API key integration.
